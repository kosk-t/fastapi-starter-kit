# FastAPI Starter Kit

このプロジェクトは、FastAPIを使用したシンプルなスターターキットです。基本的なCRUD操作を実装したAPIを提供します。

## 構成ファイル

- `app/main.py`: アプリケーションのエントリーポイント
- `app/models.py`: データベースモデルの定義
- `app/schemas.py`: リクエストおよびレスポンスのデータ構造
- `app/crud.py`: データベース操作の実装
- `app/database.py`: データベース接続の設定
- `requirements.txt`: 必要なPythonパッケージのリスト

## セットアップ手順

1. リポジトリをクローンします。
   ```
   git clone <repository-url>
   cd fastapi-starter-kit
   ```

2. 必要なパッケージをインストールします。
   ```
   ## Setup
   python -m venv env
   env\Scripts\activate
   pip install -r requirements.txt

   cd frontend
   npm install
   ```

3. アプリケーションを起動します。
   ```
   cd frontend
   npm run dev
   ```

## 使用方法

APIエンドポイントにアクセスすることで、アイテムの作成、取得ができます。

- アイテムの作成: `POST /items/`
- アイテムの取得: `GET /items/{item_id}`

詳細なAPIの使用方法は、各エンドポイントのドキュメントを参照してください。

---

### **段階1: 基本的なゲーム画面の作成**
**ゴール**: ゲームエリアとプレイヤーキャラクター（宇宙船）を画面に表示し、左右に動かせるようにする。

**Copilotへの指示**:
1. `HTML5 Canvasを使用してゲームエリアを作成するコードを生成してください。`
2. `Canvas内に青い四角形のプレイヤーキャラクターを描画してください。`
3. `Arrowキーでキャラクターを左右に動かすコードを追加してください。`

---

### **段階2: エイリアンの行列を追加**
**ゴール**: エイリアンが複数表示され、Canvas上でランダムな方向に動き始める。

**Copilotへの指示**:
1. `Canvasに5x3のエイリアン行列を描画してください。`
2. `エイリアンをランダムな方向に少しずつ移動させるコードを生成してください。`
3. `エイリアンが画面の端に到達したら反転するコードを追加してください。`

---

### **段階3: 弾丸の発射**
**ゴール**: プレイヤーがスペースキーを押すと弾丸を発射し、弾丸がエイリアンに当たるとエイリアンが消える。

**Copilotへの指示**:
1. `スペースキーで弾丸を発射するコードを作成してください。`
2. `発射された弾丸がCanvas内で上方向に移動するコードを追加してください。`
3. `弾丸がエイリアンに当たった場合にエイリアンを消すロジックを作成してください。`

---

### **段階4: スコアの表示**
**ゴール**: プレイヤーがエイリアンを倒すたびにスコアが増え、画面上部に表示される。

**Copilotへの指示**:
1. `スコアを管理する変数を作成してください。`
2. `Canvas上部にスコアをリアルタイムで描画するコードを生成してください。`
3. `エイリアンを倒すたびにスコアを増加させるロジックを追加してください。`

---

### **段階5: ゲームオーバーの条件**
**ゴール**: エイリアンが画面の下部に到達したらゲームオーバーになる。

**Copilotへの指示**:
1. `エイリアンがCanvasの下部に到達したかどうかをチェックするコードを作成してください。`
2. `ゲームオーバーの条件を満たした場合、画面に「Game Over」と表示するロジックを追加してください。`

---

### **段階6: スコアの保存機能**
**ゴール**: ゲーム終了後、スコアがバックエンドに送信される。

**Copilotへの指示**:
1. `ゲーム終了時にスコアをAPIエンドポイントに送信するコードを作成してください。`
2. `axiosを使用して、スコアをFastAPIのバックエンドにPOSTするリクエストを生成してください。`

---

### **段階7: リーダーボードの表示**
**ゴール**: 保存されたスコアをバックエンドから取得し、リーダーボード形式で画面に表示。

**Copilotへの指示**:
1. `APIからスコアリストを取得し、リーダーボードを作成するコードを生成してください。`
2. `Vue.jsでリーダーボードをテーブル形式で表示してください。`

---

### **段階8: 難易度調整**
**ゴール**: エイリアンの移動速度や弾丸発射制限を導入して、ゲームを難しくする。

**Copilotへの指示**:

以下は、スペースインベーダーにボスやストーリー性を追加するおまけ要素のアイデアです。ゲームをより面白く、やりがいのあるものにするための工夫です。

---

## **ボスの出現に関するアイデア**

1. **一定スコアでボス出現**
   - プレイヤーが一定のスコアに到達するごとにボスが出現。
   - **特徴**:
     - 通常のエイリアンよりも体力が高い。
     - 特殊な弾を撃つ攻撃をする。
     - ボス専用のアニメーションや特殊効果を追加。

2. **時間制ボスバトル**
   - 一定時間プレイすると、次のステージへ進む代わりにボスバトルが開始。
   - **特徴**:
     - 時間内にボスを倒さないとゲームオーバー。
     - ボスの攻撃パターンが段階的に激しくなる。

3. **ランダムボスエンカウント**
   - ランダムなタイミングで、ボスがステージに乱入。
   - **特徴**:
     - 他のエイリアンが退場し、ボスとの一対一の戦いに。
     - ボスを倒すと大幅なスコアアップや特殊アイテムを獲得。

---

## **ストーリー性を持たせるアイデア**

1. **バックストーリー**
   - 宇宙船のパイロットが地球を守るために戦っている設定を追加。
   - 各ステージの開始時やボス戦の前に短いストーリーを表示。
   - **例**: 
     - 地球がエイリアンに侵略されており、プレイヤーは地球の最後の希望。
     - ボスが「侵略軍のリーダー」であり、ステージごとに異なる性格や背景がある。

2. **キャラクターの成長**
   - プレイヤーキャラクターが経験値を獲得し、レベルアップする仕組み。
   - レベルアップ時に以下を選べる:
     - 弾丸の速度や威力の強化。
     - 宇宙船の移動速度向上。
     - 特殊スキル（例: シールド展開、範囲攻撃）。

3. **探索モード**
   - 通常のステージクリア型とは別に、宇宙を探索して新たな敵や仲間を見つけるモード。
   - 隠しアイテムやボスが登場する。

4. **マルチエンディング**
   - プレイヤーのスコアや選択によって結末が変化。
   - 例:
     - スコアが高い場合: プレイヤーがエイリアンを撃退して地球を守る。
     - スコアが低い場合: エイリアンが地球を侵略し、次の戦いが続く設定に。

---

## **アイテムとアップグレード**

1. **パワーアップアイテム**
   - 倒したエイリアンやボスが以下のようなアイテムをドロップ:
     - ダブルショットやトリプルショット。
     - 一定時間無敵になるシールド。
     - スコアが倍増するアイテム。

2. **ストア機能**
   - スコアやポイントを使ってアップグレードを購入:
     - 宇宙船の速度強化。
     - ボムやミサイルなど強力な武器を追加。
     - 宇宙船のデザイン変更（コスチュームのような要素）。

---

## **演出とビジュアルの工夫**

1. **特殊エフェクト**
   - ボス戦時に画面全体が点滅するなどの緊張感を演出。
   - ボスの攻撃に派手なアニメーションや音を付ける。

2. **シネマティックイントロ**
   - ステージ開始時やボス出現時に短いムービー風の演出を追加。
   - 例: 宇宙船が敵陣に突入するシーン。

3. **ユニークなボスデザイン**
   - 各ボスを個性的にする。
     - 例: 巨大なクラゲ型エイリアン、変形する機械ボス。

---

## **追加モードの提案**

1. **サバイバルモード**
   - 可能な限り長時間生き残ることを目指すモード。
   - 波状攻撃が続き、途中で強力なボスが登場。

2. **協力モード**
   - 2人プレイで協力して敵を倒すモード。
   - 一方がシールドを操作し、もう一方が攻撃を担当。

3. **タイムアタックモード**
   - 制限時間内にできるだけ多くの敵を倒してスコアを競う。

---

## **ストーリーとボスを連動させるアイデア**
1. **エピソード形式**
   - 各ステージがエピソードになっており、ボスとの対話が行われる。
   - ボス戦前に簡単なストーリーダイアログを追加。

2. **ステージテーマの変更**
   - ステージごとに背景や音楽を変更して、異なる惑星や宇宙空間を表現。

---

これらのアイデアを組み合わせることで、ゲームに奥行きやプレイヤーの没入感を高める要素を追加できます。どれを実装するかは、優先順位と開発リソース次第です！
